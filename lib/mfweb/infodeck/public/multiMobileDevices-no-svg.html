No Support for SVG

I’m afraid the infodeck on multiple mobile devices uses SVG and thus requires a browser that will display SVG. According to the browser feature detection software I'm using (modernizr) your browser does not support SVG and therefore cannot display the infodeck. IE 8 is the most probable non-SVG browser.

If possible please try again using a more recent browser. Up-to-date versions of Firefox, Chrome, and Safari certainly work well. IE 9 also displays the SVG (although does not support the animated transitions between slides).
