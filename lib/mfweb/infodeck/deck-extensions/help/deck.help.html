How to use this deck

  • You can also use a keyboard:
    • left-arrow, down-arrow, page-down, and space move forwards.
    • right-arrow, up-arrow, page-up, and backspace move backwards
    • press “g” to bring up a dialog which allows you to jump to any slide number
  • If you’re on a tablet you can swipe to move back and forth

This deck was built using deck.js
